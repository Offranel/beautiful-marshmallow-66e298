Sengele Ministries - Site Officiel
Ce projet est la page d'accueil officielle de Sengele Ministries.
C'est un site web vitrine conçu pour présenter la mission du ministère (prière, enseignement et espérance)
, afficher les horaires des services en ligne, promouvoir le programme mensuel et permettre aux fidèles d'entrer en contact.

  Fonctionnalités
Bannière Dynamique (Header) : Un script JavaScript fait défiler automatiquement les images d'arrière-plan du header toutes les 3 secondes.

Navigation Intuitive : Une barre de navigation ancrée permettant de se déplacer rapidement dans les sections de la page.

Carrousel de Galerie : Intégration de la bibliothèque Slick Carousel pour afficher les photos de la galerie sous forme de diaporama automatique (défilement de 3 images simultanées).

Formulaire de Contact Fonctionnel : Un formulaire lié à FormSubmit.co pour recevoir directement les messages des utilisateurs par e-mail.

Intégration Vidéo : Lien direct vers les partages et extraits vidéo (YouTube Shorts).

Design Responsive & Épuré : Palette de couleurs soignée (fond beige, textes navy et accents avec des boutons thématiques).

🛠️ Technologies Utilisées
HTML5 : Structuration du contenu et des textes bibliques.

CSS3 : Styles personnalisés (police, mise en page, boutons, gestion du responsive de la galerie).

JavaScript (Vanille) : Logique personnalisée pour le changement automatique du fond d'écran du header.

jQuery (v1.12.4) : Requis pour le fonctionnement du carrousel.

Slick Carousel (v1.8.1) : Plugin pour la mise en place du carrousel d'images dynamique.

Structure du Code Principal
index.html : Contient l'ensemble du contenu, la structure de la page, le formulaire, ainsi que les scripts d'animation.

style.css : Feuille de style externe (gère la mise en page globale en complément des styles internes).

images/ : Dossier contenant les ressources visuelles du site (ex: TUEQ6134.JPG, IMG_7497.JPG, etc.).
